--------------------
Instrucciones de uso
--------------------

# Para cambiar el dataset de prueba
1) Modifique el parámetro de la función generador del archivo dataset.py
2) Ejecute el programa
3) El archivo dataset.txt se modificará automáticamente en la carpeta dataSets

# Para correr el programa 
1) Compile el archivo usando el comando make desde la carpeta LAB2
2) Para ejecutar, ./output
3) Por pantalla se mostrarán los tiempos de ejecución en microsegundos
